There are two ways to call a message.
1:
$('#allBase').anymessage({'message':'All your base are belong to us'});
This is the same style, using an API response for the message contents:
$('#allBase').anymessage('message':responseData);
(reponseData could be formatted using _err, _msgs or @issues response format)
